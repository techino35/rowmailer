# Privacy Policy for RowMailer

**Last Updated: 2026-03-22**

RowMailer stores all settings locally in your spreadsheet's document properties. No data is ever sent to external servers. Email sending uses your own Google account via the Gmail API.

| Scope | Purpose |
|-------|---------|
| `spreadsheets` | Read spreadsheet data |
| `gmail.send` | Send emails |
| `script.scriptapp` | Create time-based triggers |
| `userinfo.email` | Identify current user |

*RowMailer is not affiliated with Google LLC.*
